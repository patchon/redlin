Patch report

A web front displaying patch reports for servers

Project Details

  • Client Former Employer
  • What we did Creating a patchreport for servers
  • Language Bash
  • Techniques SNMP
  • Year 2014
  • Lines of code ~ 1500

A weekly summary of downloaded and applied patches

The aim of this project was to visualize a summary (patchreport) of which patches that had been applied for which environment. Developers and managers could then easily keep track of where certain patches were applied. Data was collected through snmp and compared with the latest available patches.
